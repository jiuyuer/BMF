artTabs 配置参数演示

1. 菜单配置

调用此方法之后，完成如下几点功能：

  • 1). 选择导航效果的绑定
  • 2). 绑定menu上的菜单新增功能
  • 3). 绑定tab上list的切换
  • 4). 绑定tab上list里面a标签的关闭
  • 5). 高度自适应(主要是iframe的外围)
  • 6). 左右键tab切换
  • 7). 绑定tab上list的右键菜单及功能
        var menuData = [{
                text : '样式说明',
                items : [
                    {id:'tabsMethod',text:'tab调用方法说明',href:'demoPage/tabs.html'},
                    {id:'test2',text:'测试页面二',href:'demoPage/test2.html'}
                ]
                },{
                text : '测试',
                items : [
                    {id:'test3',text:'测试页面三',href:'demoPage/test3.html'}
                ]
                }
        ]
        
        /*调用*/
        $.artTabs({
            menu : menuData
        })

2. 关闭当前tab页

closeTab，默认参数为 false
调用此方法后，内容页面展示的是当前关闭窗口的前一个tab页面 

        $.artTabs({
            closeTab : true
        })

3. 刷新当前tab页

isRefresh，默认参数为 false
调用此方法后，会刷新当前窗口

tip：1. 配合修改（或新增），跳转功能来调用居多
2. 很少单独调用此方法，在list的右键上已绑定此功能。 

        $.artTabs({
            isRefresh : true
        })

4. 新增（或修改）tab页

调用方法：addTab
以一个对象字面量items的形式传入对应的参数，包括 id，title，url

tip：1. 如果新增的窗口已打开，那么，将直接跳转到相应的窗口并显示 2. 如果是修改方法（在表格修改中比较多），一般是跳转并刷新跳转的页面 

        $('#button').artTabs({
            addTab: {
                    items: {
                            id: 'test4',
                            title: '测试页面四',
                            url: '/demoPage/test4.html'
                    }
            }
            //isRefresh : true
        })

5. 跳转到对应tab页

调用方法：toTab
传入对象id，此id为跳转到对应窗口的id值

tip： 如果想要跳转的页面不存在（id值存在），那么就会弹出提示并返回 

    	$('#button2').artTabs({
            toTab: {id : "test2"},
            //closeTab: true,
            isRefresh: true
        })
